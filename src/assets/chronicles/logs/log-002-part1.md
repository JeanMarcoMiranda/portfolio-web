# Arquitectura Híbrida: Diario de Desarrollo

## Capítulo 1: El Concepto Inicial

Cuando decidí construir este portafolio, quería algo más que una simple página web. Quería una **experiencia**.

La idea era combinar:
1. La usabilidad de una SPA moderna (Single Page Application) usando **Angular**.
2. La estética inmersiva de los RPG retro (Role Playing Games).
3. Elementos de interfaz de terminales cyber-punk.

### Retos Iniciales

El principal desafío fue asegurar que el uso intensivo de colores oscuros, fuentes pixeladas (`Press Start 2P`) y animaciones no comprometiera el rendimiento ni la accesibilidad.

En el próximo capítulo, exploraremos cómo estructuré los tokens de diseño en SCSS para mantener todo bajo control.
